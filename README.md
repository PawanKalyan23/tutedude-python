# Tutedude Python Assignment

## Assignment Tasks

### 1. Grade Checker

#### Description

This program takes a student's score as input and displays the corresponding grade using if-elif-else statements.

#### Code File

`grade-checker.py`

#### Sample Output

```text
Enter your score: 95
Grade: A
```

---

### 2. Student Grades

#### Description

This program uses a dictionary to store student names and grades. It allows the user to add a student, update a student's grade, and display all student records.

#### Code File

`student-grades.py`

#### Sample Output

```text
Enter student name: Rahul
Enter grade: A
Enter student name to update: Rahul
Enter new grade: B
Grade updated successfully.

Student Grades:
Rahul : B
```

---

### 3. Write to a File

#### Description

This program creates a text file and writes content to it using Python file handling functions.

#### Code File

`write-file.py`

#### Sample Output

```text
Content written to output.txt successfully.
```

#### Content Written to File

```text
Hello, this is my first file handling program.
```

### 4. Read from a File

#### Description

This program reads content from a text file and displays it on the screen.

#### Code File

`read-file.py`

#### Sample Output

```text

```
